# 🌎 Around The U.S.

## 📌 Description

**Around The U.S.** is an interactive web application built with **React**. It lets users view and like photos, and interact with forms in a simple and intuitive way.

The interface focuses on user experience, using a **responsive design** that adapts to different screen sizes. It provides a nice navigation on both computers and mobile devices.

The goal of this project was to practice fundamental concepts of modern front-end development, such as components, state management, and creating dynamic, reusable interfaces.

---

## ✨ Features

* ❤️ Like photos and interact with the page content;
* 📝 Fill out and submit forms directly through the interface;
* 📱 Fully responsive layout for different devices;
* ⚡ Dynamic updates of the page elements;
* 🎨 Modern and simple interface.

---

## 🛠️ Technologies Used

* **React** → Creating reusable components and managing state;
* **JavaScript (ES6+)** → Application logic and interactivity;
* **JSX** → Structuring components;
* **HTML5** → Semantic structure of the application;
* **CSS3** → Modern and responsive styling.

## 🚀 How to run the project

### 1. Clone the repository

```bash
https://github.com/MarcelaMourar/web_project_around_react.git
```

### 2.Go to the project folder

```bash
cd web_project_around_react
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Open the project in your browser

The application will be available at:

```text
[http://localhost:3000]
```
