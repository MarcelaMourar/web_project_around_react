# 🌎 EUA Afora

## 📌 Descrição

**EUA Afora** é uma aplicação web interativa desenvolvida com **React**, que permite aos usuários visualizar, curtir fotos e interagir com formulários de maneira simples e intuitiva.

A interface foi construída com foco na experiência do usuário, utilizando um **design responsivo** que se adapta a diferentes tamanhos de tela, proporcionando uma navegação agradável tanto em computadores quanto em dispositivos móveis.

Este projeto teve como objetivo praticar conceitos fundamentais do desenvolvimento front-end moderno, como componentização, gerenciamento de estado e criação de interfaces dinâmicas e reutilizáveis.

---

## ✨ Funcionalidades

* ❤️ Curtir fotos e interagir com o conteúdo da página;
* 📝 Preencher e enviar formulários diretamente pela interface;
* 📱 Layout totalmente responsivo para diferentes dispositivos;
* ⚡ Atualização dinâmica dos elementos da página;
* 🎨 Interface moderna e intuitiva.

---

## 🛠️ Tecnologias Utilizadas

* **React** → Criação de componentes reutilizáveis e gerenciamento de estado;
* **JavaScript (ES6+)** → Lógica da aplicação e interatividade;
* **JSX** → Estruturação dos componentes;
* **HTML5** → Estrutura semântica da aplicação;
* **CSS3** → Estilização moderna e responsiva.

## 🚀 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/eua-afora.git
```

### 2. Acesse a pasta do projeto

```bash
cd eua-afora
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Inicie o servidor de desenvolvimento

```bash
npm run dev
```

### 5. Abra o projeto no navegador

A aplicação estará disponível em:

```text
http://localhost:5173
```

> A porta pode variar dependendo da configuração do projeto.
